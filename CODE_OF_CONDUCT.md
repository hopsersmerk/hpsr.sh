# Código de conducta

## Nuestro compromiso

Con el interés de fomentar una comunidad abierta y respetuosa, quienes participan en `hpsr.sh` se comprometen a hacer de la colaboración una experiencia libre de acoso para todas las personas.

## Nuestros estándares

Ejemplos de comportamiento que contribuyen a un ambiente positivo:

1. Usar un lenguaje respetuoso y profesional
2. Aceptar feedback técnico de buena fe
3. Centrarse en mejorar el proyecto y ayudar a otras personas
4. Mostrar empatía con personas que están aprendiendo
5. Discutir ideas con claridad sin ataques personales

Ejemplos de comportamiento inaceptable:

1. Ataques personales
2. Insultos o descalificaciones
3. Publicación de información privada de otras personas
4. Acoso público o privado
5. Comentarios discriminatorios o degradantes

## Responsabilidades de mantenimiento

Quienes mantienen el proyecto pueden moderar, editar o eliminar comentarios, contribuciones o interacciones que violen este código de conducta.

## Alcance

Este código aplica en:

1. Issues
2. Pull requests
3. Discussions
4. Releases
5. Cualquier otro espacio oficial del proyecto

## Reportes

Si necesitas reportar una conducta inapropiada, puedes contactar a:

`contacto@hpsr.sh`

Los reportes serán revisados y tratados de forma razonable y confidencial según el contexto.

## Atribución

Este documento está inspirado en Contributor Covenant, adaptado al contexto de este proyecto.
